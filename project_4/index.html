knuckle_dragger

knuckle_dragger is a component you can use to identify thick-headed users on your web page. Upon detection, knuckle_dragger makes everylink go to the same place, provoking the user to smash his computer. Cookies track him, and, by-golly, you can find him, and sell him a faulty computer that has more knuckle-dragger tricks in its bag.

  • has only one function
